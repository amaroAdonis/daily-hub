# Especificações de features

Cada feature ganha um documento próprio seguindo o
[template](./_template.md). Preenchido ao concluir a fase correspondente.

| Feature      | Fase | Documento                |
| ------------ | ---- | ------------------------ |
| Tarefas      | 1    | _tasks.md_ (em breve)    |
| Agenda       | 2    | _calendar.md_ (em breve) |
| Compromissos | 3    | _events.md_ (em breve)   |
| Metas        | 4    | _goals.md_ (em breve)    |
| Anotações    | 5    | _notes.md_ (em breve)    |
| Contatos     | 6    | _contacts.md_ (em breve) |
| Integração   | 7    | _links.md_ (em breve)    |
